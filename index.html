UIHex

Instantly convert a hex color code to a UIColor method.

Show me tips. By Eugene Belinski. Released under GNU-GPL. Send suggestions here. Fork this website here.

  • You can also get the UIColor code by entering the hex code in the url like this: uihex.com/#fafafa. (credit)
  • You can enter shorthand three digit hex codes, such as #123.
  • It doesn't matter whether or not you include the # symbol in the hex code box.
